   <h2 align="center">Baixando o projeto</h2>

   ```

   - Fazendo o clone:
   $ git clone https://github.com/Ljuniorcode/discovery-Proffy.git


   - Instalar as dependencies:
   $ npm install


   - Execute o server:
   $ npm run dev

   - Abra o navegador e digite:
   http://localhost:5000/

   ```
